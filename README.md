# 便签墙项目 (Sticky Notes Wall)

## 项目概述
这是一个用纯 HTML、CSS 和 JavaScript 实现的便签墙应用。用户可以看到一个网格背景上随机生成的彩色便签卡片，每张卡片显示一条温馨的提示信息。卡片可以进行拖拽、最小化、最大化和关闭操作，提供了类似窗口管理的交互体验。

## 技术架构
- **前端**: 纯 HTML5, CSS3, JavaScript (ES6+)
- **布局**: 使用 CSS Grid 和 Flexbox 进行布局
- **交互**: 使用 Pointer Events API 处理鼠标和触摸事件
- **动画**: 使用 CSS transitions 和 JavaScript requestAnimationFrame 实现流畅动画
- **响应式设计**: 针对移动端和桌面端进行了适配

## 主要功能
1. **便签生成**: 自动在网格背景上生成带温馨信息的便签卡片
2. **窗口控制**: 每张卡片都有关闭、最小化、最大化的窗口控件
3. **拖拽**: 可以通过拖拽头部移动卡片位置
4. **最大化/还原**: 双击头部或点击最大化按钮可全屏显示卡片
5. **响应式**: 根据设备类型（移动/桌面）自动调整卡片数量和生成频率

## 文件结构
- `index.html`: 包含完整的 HTML 结构、CSS 样式和 JavaScript 逻辑

## 开发约定
- 使用现代 CSS 特性如 `clamp()`, `dvh`, `pointer: coarse` 媒体查询
- 使用 WeakMap 存储卡片状态，避免 DOM 元素上的额外属性
- 自动清理超出数量限制的旧卡片以优化性能
- 考虑了移动设备上的用户体验和性能优化

## 运行方式
直接在浏览器中打开 `index.html` 文件即可运行。由于这是一个静态 HTML 页面，不需要额外的服务器环境。
